HTML and CSS Project: Online Vignette and Third-Party Liability Insurance Checker
Overview

This project is a simple web application designed to check the validity of an online vignette and Third-Party Liability Insurance by entering the registration number of a car. The user-friendly interface is implemented using HTML and CSS, making it accessible and intuitive for users to perform quick checks on the go.
Features

    Vignette Checker: Enter the registration number of a vehicle to verify the status of its online vignette.
    Insurance Checker: Validate the Third-Party Liability Insurance of a vehicle by providing its registration number.
    Responsive Design: The HTML and CSS ensure a responsive and visually appealing layout, adapting to various screen sizes for a seamless user experience.

How to Use

    Clone the repository to your local machine.

    

    git clone https://github.com/VladislavHristov/VladislavHristov.github.io

    Open the index.html file in a web browser to access the application.

    Enter the registration number of the vehicle in the designated input field.


Technologies Used

    HTML: The backbone of the project, providing the structure of the web pages.
    CSS: Styling elements to ensure a visually appealing and user-friendly interface.
